# Simple Chat - Лёгкий мессенджер на Next.js

💬 Простой и быстрый чат без регистрации с современным дизайном.

## ✨ Особенности

- 🎨 **Современный дизайн** - Glassmorphism, градиенты, анимации
- 🔐 **Без регистрации** - просто введите имя
- 👥 **Личные и групповые чаты** - создавайте чаты с несколькими участниками
- 📷 **Отправка изображений** - делитесь картинками
- 😊 **Смайлики** - встроенная панель эмодзи
- ✍️ **Индикатор набора** - показывается когда пользователь печатает
- 💾 **localStorage** - данные хранятся в браузере
- 🚀 **Быстрый запуск** - готов к деплою за 5 минут

## ⚠️ Важное замечание

**Этот чат использует localStorage и работает только в рамках одного браузера!**

✅ **Подходит для:**
- Демонстрации и обучения
- Личного использования в одном браузере
- Тестирования концепций

❌ **Не подходит для:**
- Общения между разными устройствами
- Публичного мессенджера
- Коммерческого использования

Для полноценного мессенджера с базой данных см. ветку `with-database`.

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
npm install
```

### 2. Запуск в режиме разработки

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000)

### 3. Сборка для production

```bash
npm run build
npm run start
```

## 📦 Деплой

### Docker

```bash
# Собрать образ
docker build -t simple-chat .

# Запустить контейнер
docker run -p 3000:3000 simple-chat
```

### VPS (TimeWeb, DigitalOcean, etc.)

Подробная инструкция в [DEPLOYMENT.md](./DEPLOYMENT.md)

Кратко:
```bash
# Установить Node.js и PM2
npm install -g pm2

# Установить зависимости
npm install

# Собрать проект
npm run build

# Запустить
pm2 start npm --name "simple-chat" -- start
```

## 🛠 Технологии

- **Next.js 14** - React фреймворк
- **TypeScript** - типизация
- **Tailwind CSS** - стилизация

## 🎯 Как использовать

1. Откройте http://localhost:3000
2. Введите ваше имя
3. Откройте эту же ссылку в новой вкладке
4. Введите другое имя
5. Создайте чат между пользователями
6. Общайтесь! 💬
